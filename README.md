## Donzel Asira | Full-Stack Web Developer
A high-performance portfolio merging technical logic with creative soul. This project showcases my journey as a Computer Engineer 
and my work delivering high-scale solutions.

## Features
- Dual-Mode Experience: Toggle between a Professional Hub (Work, Education, Journey) and a Personal World.
- Cinematic Transitions: Smooth, blur-filtered page wipes powered by Framer Motion.
- Bilingual Support: Full English (EN) and German (DE) localization.
- Parallax Project Cards: 3D tilt effects and mouse-tracked parallax imagery.


## Tech Stack
### Frontend & Design
- **Next.js / React 18** + **TypeScript**
- **Tailwind CSS** (Advanced Glassmorphism & custom animations)
- **Framer Motion** (3D tilt interactions & layout animations)
- **Lucide React** (Minimalist iconography)

### Infrastructure
-  **Render** (Deployment)
-  **Git** (Version Control)
